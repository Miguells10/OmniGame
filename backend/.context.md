# 🧠 Backend Context Mapper

## 🏗 Arquitetura Atual
- **Boot:** Spring Boot 3.4 (Java 21)
- **Padrão Arquitetural:** Arquitetura Hexagonal (Domain, Application, Infrastructure, Presentation)
- **Banco de Dados:** PostgreSQL via Supabase Cloud
- **Modelagem de Dados:** EAV (Entity-Attribute-Value) para lidar com metadados estritamente dinâmicos entre diferentes instâncias de jogos.
- **Motor de IA (RAG):** Spring AI acoplado com `pgvector` operando busca semântica por similaridade de cossenos nas bases de conhecimento.
- **Segurança da API:** Autenticação totalmente Stateless. Filtro de segurança baseado em Token JWT validado com a *secret* assimétrica do painel do Supabase Auth.
- **Documentação:** Swagger UI / OpenAPI 3 live documentation (`/swagger-ui.html`).
